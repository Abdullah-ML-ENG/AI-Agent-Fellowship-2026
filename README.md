## 📦 Project Overview

This project is a modern browser-based **AI Workspace** built with HTML/CSS/JavaScript.  
It supports:

- Multi-session chat interface
- Provider selection (**Groq** / **Gemini**)
- Dynamic model selection
- API key input (session memory)
- Prompt templates (built-in + custom)
- Voice typing (Web Speech API compatible browsers)
- Local auth screen for dashboard-style access
- Theme toggle (dark/light)

## 📁 Repository Structure

- `index.html` — Main source code (UI + logic in one file)
- `README.md` — Project and profile documentation
- `requirements.txt` — Runtime requirements/dependencies
- `INSTALLATION_GUIDE.md` — Setup and usage instructions

## 🔐 Security Notes

- Do **not** hardcode API keys in source code.
- Use environment-level secret handling for production deployments.
- This project keeps keys in tab memory during active use.
